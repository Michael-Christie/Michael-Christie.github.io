DragonCraft

Programmer | 2021 - Now

Key Responsibilities

  • Optimised, restructed and redesigned infinite world generation systems
  • Implemented A/B tests for new features to increase retention within the game
  • Updated UI, resulting in optimised draw calls and better visuals
  • Implement new Worlds to host engaging minigames in, and added editor functionality for designers to create new worlds with
  • Reduce ANR's and Crashes to improve user experience
