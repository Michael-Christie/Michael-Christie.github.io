Dungeon Crawler

Dungeon Crawler is a current ongoing project. The project is currently in early stages. The idea behind the project is creating a first person, deck building, exploration and adventure dungeon crawler.

Current State

Dungeon Crawler has been going through continuous level design prototyping and experimenting over the last year. The game currently has a main menu with player selector, hub and rough early stages level mockup and a demo can be found here. The levels and art are currently being redeveloped based on the experimenting I've been doing over the last year. You can check out the code and backend here.
